FranceJS IRC bot
============
This is the IRC bot of the french Javascript group. It logs our IRC chan ( irc://irc.freenode.net/francejs )
and obey to some commands. Feel free to complete him !


Getting started
------------
npm install irc; echo 'Enjoy'
